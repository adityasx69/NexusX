# 🚀 PBL Project

A modern web application built using **React + Vite**, designed for performance, scalability, and clean UI.

---

## 📌 Features

* ⚡ Fast and optimized with Vite
* 🎨 Responsive and clean UI
* 🔥 Modern React (hooks-based architecture)
* 📦 Modular and scalable folder structure
* 🌐 Ready for deployment (Vercel / GitHub Pages)

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Styling:** CSS / Tailwind (if used)
* **Deployment:** Vercel

---

## 📂 Project Structure

```
PBL/
```

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/Shiv-9-zz/PBL.git
cd PBL
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

---

## 🏗️ Build for Production

```
npm run build
```

Preview build:

```
npm run preview
```

---

## 🚀 Deployment

This project is deployed using **Vercel**.

To deploy:

1. Push your code to GitHub
2. Import the repo in Vercel
3. Click **Deploy**

---

## 🐛 Known Issues

* Dependency conflicts may occur with newer Vite versions
  👉 Recommended: Use `vite@7` for stability
